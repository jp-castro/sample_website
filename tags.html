Important HTML tags

  • This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

  • The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1‐h6 headings include a heading (levels 1‐6)
p paragraph include a paragraph of text
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> at least a billion people
a anchor make hyperlink <a href="http://learnenough.com/">Learn Enough</a> Learn Enough
img image include an image <img src="https://bit.ly/1MZAFuQ" alt="Michael Hartl">
code code display code <code><a href="http://www.apple.com"></a></code> code block

Document Tags

Tag Name Purpose Example Result
html html language to render webpages <html><head><body></body></head>/html> {enter web page here}
head head tag to contain all head elements <head></head> Webpage header
body body tag to contain the main website elements <body>The content of the document......</body> The content of the document......
title title tag to contain the title of the web page <title>Webpage tab title</title> {insert page title here}
meta meta tag to provide metadata about HTML document <meta charset="utf-8"> {insert metadata here}

Styling Elements

Tag Name Purpose Example Result
style style add 'styles' to content within html tags <a href="tags.html" style=" text-decoration: none; color: #0000ff;">HTML Tags</a> HTML Tags
ul ul create a bulleted list <ul><li>first item</li><li>second item</li></ul>
  • first item
  • second item
ol ol create a numbered list <ol><li>first item</li><li>second item</li></ol>
  1. first item
  2. second item
div div tag to define a section of a webpage <div>this is a section of a webpage</div>
It may not look it, but this text is "div'd"
span span tag to group inline elements in a webpage <span>span example</span> It may not look it, but this text is 'spanned'
